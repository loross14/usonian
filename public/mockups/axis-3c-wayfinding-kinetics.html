  • Scroll progress: Fixed bar at top, width = scroll %
  • Section dots: Active dot scales up + fills red
  • Staggered reveals: Combined X/Y offset entry (e.g., -48px X, +16px Y)
  • Reveal directions: left, right, bottom with 80ms stagger

  • Supports CSS scroll-driven animations for modern browsers
